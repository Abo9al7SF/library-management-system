# Library Management System

A Java-based application designed to manage library operations, including book inventory, member registration, and the tracking of borrowed materials. This project focuses on efficient data management and the application of core Object-Oriented Programming (OOP) principles.

## Key Features
- Inventory Management: Track book details such as title, author, ISBN, and availability.
- Member Services: Manage library memberships and user profiles.
- Circulation Logic: Automated system for borrowing and returning books with due-date tracking.
- Search Functionality: Ability to filter and find books within the library database.

## Tech Stack
- Language: Java
- Paradigm: Object-Oriented Programming (OOP)
- Tools: Git, GitHub

## OOP Principles Applied
- Inheritance: Used a base 'Item' class for common library assets, extended by specific types like Books or Magazines.
- Encapsulation: Private data fields for sensitive member and book information, accessed via public methods.
- Polymorphism: Overridden methods for handling different types of library transactions.
- Abstraction: Defined clear interfaces for system behaviors like 'Searchable' or 'Borrowable'.

## Project Structure
- src/: Java source files (.java)
- docs/: Design documentation and system reports.
